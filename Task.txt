Завдання №1  

Створіть потік, який "приймає" в себе колекцію елементів і
викликає для кожного елемента колекції метод ToString(),
після чого виводить результат роботи методу на екран.  

Завдання №2  

Створіть клас Bank, в якому будуть наступні властивості: 
 -	int money, 
 -	string name, 
 -	int percent. 
Побудуйте клас так, щоб при зміні однієї з властивостей 
створювався новий потік, який записував би дані про 
властивості класу в текстовий файл на жорсткому диску. 
Клас повинен інкапсулювати в собі всю логіку багатопоточності.
